# トシヤイズムスロット V2 抽選・内部状態遷移図

この図は `src/v2/slot-v2-rules.js` と `src/v2/slot-v2-engine.js` の現在実装を基準にしたもの。
演出用の `normalStage` は内部状態そのものではなく、通常時動画の出やすさだけを変える示唆パラメータとして扱う。

## 1. 通常時 1G の抽選フロー

```mermaid
flowchart TD
  A["通常時レバー"] --> B["3枚BET<br/>リプレイ後はBET免除"]
  B --> C["内部役抽選"]
  C --> D["停止方式を決定<br/>ベル/リプレイ/チャンスは制御停止<br/>スイカ/チェリーは目押し払い出し"]
  D --> E["リール停止操作"]
  E --> F["5ライン/左チェリー判定<br/>上段・中段・下段・右下がり・右上がり"]
  F --> G{"払い出し成立?"}
  G -->|はい| H["払い出し<br/>ベル8枚 / スイカ6枚 / チェリー2枚 / チャンス1枚"]
  G -->|いいえ| I["払い出しなし<br/>偶然の斜め揃いも出さない"]
  H --> J["内部役効果処理"]
  I --> J
  J --> K["内部状態移行抽選"]
  K --> L["normalStage 示唆更新<br/>内部状態とは非連動"]
  L --> M{"前兆に入った?"}
  M -->|はい| N["前兆 16-32G<br/>pendingBonus 確定済み"]
  M -->|いいえ| O["通常継続"]
```

### 役抽選

| 役 | 確率 | 払い出し | 備考 |
| --- | ---: | ---: | --- |
| ベル | 1/40 | 8枚 | 通常時は内部成立したら必ず有効ラインへ自動停止。揃わない停止形は内部的にもハズレ |
| リプレイ | 1/7.3 | 0枚 | 通常時は内部成立時に有効ラインへ自動停止 |
| スイカ | 1/119 | 6枚 | 通常時は目押し成功時だけ払い出し。内部効果は失敗しても有効 |
| 角チェリー | 1/135 | 2枚 | 通常時は左リールにチェリーを出せば払い出し。右リールは不要。内部効果は失敗しても有効 |
| トシヤチャンス | 1/180 | 1枚 | 通常時は内部成立時に有効ラインへ自動停止。ロゴ抽選を少し優遇 |
| 中段チェリー | 1/247 | 2枚 | 通常時は左リールにチェリーを出せば払い出し。高確なら停止失敗でも前兆確定 |
| ハズレ | 残り約81.26% | 0枚 | 斜めを含む偶然揃いと左チェリーを避ける。stageだけ微変動あり |

通常時は、スイカ/角チェリー/中段チェリーだけ停止操作で払い出しが変わる。内部状態移行、前兆抽選、stage更新は停止失敗でも内部役どおりに処理する。

## 2. 内部状態の全体遷移

```mermaid
stateDiagram-v2
  [*] --> normal

  low --> low: ハズレ/ベル/リプレイ/非当選
  low --> normal: 角チェ10% / スイカ25% / チャンス20% / 中段チェ50%
  low --> high: スイカ3% / チャンス5%
  low --> prelude: チャンス1% / 中段チェ25%

  normal --> normal: ハズレ/ベル/リプレイ/非当選
  normal --> high: 角チェ12% / スイカ30% / チャンス25% / 中段チェ25%
  normal --> prelude: スイカ5% / チャンス3% / 中段チェ25%

  high --> high: ハズレ/ベル/リプレイ/非当選
  high --> prelude: 角チェ15% / スイカ25% / チャンス25% / 中段チェ100%

  prelude --> prelude: 16-32Gを消化
  prelude --> bonusReady: 残り0G
  bonusReady --> bonus: 次レバーでボーナス開始

  bonus --> bonus: 30G消化後、継続なら次SET
  bonus --> normal: 非継続なら終了
```

## 4. 前兆からボーナス開始まで

```mermaid
flowchart TD
  A["前兆突入"] --> B["pendingBonus を先に決定"]
  B --> C["入賞図柄抽選<br/>通常7 or トシヤロゴ"]
  C --> D["継続率抽選<br/>66 / 79 / 84 / 89%"]
  D --> E["保証ストック抽選"]
  E --> F["前兆 16-32G"]
  F --> G["bonusReady<br/>画面上は確定"]
  G --> H["7/ロゴ揃えターン<br/>3枚BET"]
  H --> I{"目押し成功?"}
  I -->|成功| J["ボーナス開始<br/>30G固定へ"]
  I -->|失敗| H
```

### 入賞図柄・ストック・継続率

| 図柄 | 基本比率 | 継続率振り分け | ストック振り分け |
| --- | ---: | --- | --- |
| 通常7揃い | 94 | 66%:55 / 79%:30 / 84%:11 / 89%:4 | 1個:70 / 2個:24 / 3個:6 |
| トシヤロゴ揃い | 6 | 66%:10 / 79%:35 / 84%:35 / 89%:20 | 2個:30 / 3個:28 / 4個:20 / 5個:12 / 7個:7 / 10個:3 |

補足:
- トシヤチャンス成立時はロゴ比率に +10。
- 中段チェリー成立時はロゴ比率に +12。
- ロゴ揃いは高継続率と大量ストックに寄るが、完走確定ではない。

## 5. ボーナス中の流れ

```mermaid
flowchart TD
  A["ボーナス開始"] --> B["開始時点で当該SETの継続可否を内部決定"]
  B --> C["保証ストックがあれば消費して継続確定"]
  B --> D["保証ストックなしなら継続率で抽選"]
  C --> E["30G固定消化"]
  D --> E
  E --> F["ベル / リプレイ 約50:50<br/>内部抽選で自動停止"]
  F --> G["30G到達"]
  G --> H{"内部継続?"}
  H -->|継続| I{"復活演出対象?"}
  I -->|通常継続| J["継続演出"]
  I -->|復活| K["敗北演出 -> 無音/暗転 -> ロゴ復活"]
  J --> L["次SETへ"]
  K --> L
  H -->|非継続| M["終了して通常へ"]
```

### ボーナス継続判定

| 項目 | 内容 |
| --- | --- |
| 1SET | 30G固定 |
| ボーナス中役 | ベル50 / リプレイ50。毎G内部抽選で決定 |
| 払い出し | 目押し不要。内部成立で8枚 |
| 継続率 | 66% / 79% / 84% / 89% |
| ストック | ある場合は1個消費して継続保証 |
| 復活 | 継続かつストックなしの一部。実装値は約15% |
| 終了 | 非継続なら `normal` へ戻る |

## 6. normalStage 示唆パラメータ

`normalStage` は `street / heat / deep` の3段階。内部状態を直接変えず、通常時動画の抽選重みだけを変える。

```mermaid
stateDiagram-v2
  [*] --> street
  street --> heat: レア役・ベル・ハズレなどで上昇抽選
  heat --> deep: レア役・jumpDeep抽選
  deep --> heat: リプレイなどで下降抽選
  heat --> street: リプレイなどで下降抽選
  street --> deep: jumpDeep抽選
  heat --> heat: 維持
  deep --> deep: 維持
  street --> street: 維持
```

### stage更新確率

| 役 | 上昇 | 下降 | deep直行 |
| --- | ---: | ---: | ---: |
| ハズレ | 3% | 8% | 0.5% |
| ベル | 5% | 6% | 1% |
| リプレイ | 5% | 45% | 0.5% |
| 角チェリー | 45% | 6% | 8% |
| スイカ | 58% | 4% | 14% |
| トシヤチャンス | 65% | 3% | 18% |
| 中段チェリー | 78% | 2% | 26% |

重要:
- `normalStage` は内部状態と必ずリンクしない。
- レア役でも上がらないことがある。
- リプレイでも下がらないことがある。
- ハズレやベルでも上がることがある。

## 7. 通常時動画の選択

通常時動画は回転ボタンごとには切り替えない。動画が1周して `ended` したタイミングで、現在の `normalStage` を見て再抽選する。

```mermaid
flowchart TD
  A["通常時動画再生中"] --> B["動画1周終了"]
  B --> C{"normalStage"}
  C -->|street| D["静かな動画 90%<br/>ざわつき動画 10%"]
  C -->|heat| E["ふさわしい気配 50%<br/>通常気配 40%<br/>ふさわしくない気配 10%"]
  C -->|deep| F["ふさわしい気配 50%<br/>通常気配 40%<br/>ふさわしくない気配 10%"]
  D --> A
  E --> A
  F --> A
```

例:
- heat: `normal_heat_comments` 50%、`normal_street_room` 40%、`normal_deep_rumble` 10%。
- deep: `normal_deep_rumble` 50%、`normal_street_room` 40%、`normal_heat_comments` 10%。
- `bonusReady` やボーナス開始は動画終了を待たず即切り替え。
