# テスターさんへのお願い

`パチスロ トシヤイズム` は、ブラウザだけで動くパチスロ風ミニゲームです。
実機の完全再現ではなく、配信ネタ用のオリジナル機種として触ってください。

## 触ってほしいところ

- `Left Shift` で回す。`Space` でも回せて、回転中は左・中・右の順押しで止まる。
- 弱チェリーと強チェリーの見分けが自然か見る。
  - 中段チェリー: 強チェリー
  - 上段・下段チェリー: 弱チェリー
- ベル、Tリプレイ、トシヤチャンス、ISM目の停止形に違和感がないか見る。
- 前兆からボーナス確定、ボーナス開始までの流れが分かりやすいか見る。
- 通常、チャンス、高確、前兆、確定、BB中の演出表示が意図とズレていないか見る。
- BBの継続率、1セットの払い出し、終了後の戻り方に変な印象がないか見る。
- 全体モードの説明シートが読めるか、内容が実際の遊びと矛盾していないか見る。
- あいことばの発行・復元、ショップ購入が壊れていないか見る。

## 操作

| キー | 内容 |
| --- | --- |
| `Left Shift` | 回す / bonusReady中はBB開始 |
| `Space` | 回す / 回転中は左・中・右の順押し停止 |

## フィードバックで特にほしいこと

- 「スロットとしてここは気持ち悪い」
- 「成立役と出目が合っていない」
- 「強弱や期待度が見た目で伝わらない」
- 「演出の順番、前兆、確定の見せ方が不自然」
- 「説明シートの表現が実際の挙動と違う」
