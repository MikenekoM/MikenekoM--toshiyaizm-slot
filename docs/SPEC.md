# 初期実装仕様

## 技術選定

HTML/CSS/JavaScriptの単体構成で作る。無料の静的サイト公開に向け、ビルド不要で `index.html` をそのまま配信できる形にする。

## ゲーム進行

1. プレイヤーは「回す」を押す
2. 3枚のメダルを消費する
3. その時点で内部抽選を行う
4. リールが回転する
5. 停止ボタンでリールを順番に止める
6. 抽選結果に応じて図柄と演出を出す
7. 当たりならメダルを付与する

メダルが3枚未満の場合は回せない。

## 抽選

| 結果 | 確率 | 演出 | 獲得 |
| --- | ---: | --- | ---: |
| lose | 78% | 通常コメント | 0 |
| normal | 17% | コメントざわつき、短いカットイン | 100 |
| rush | 4.5% | 金文字、決め台詞 | 800 |
| premium | 0.5% | 暗転、虹風演出、専用台詞 | 5000 |

## リール

初期版は8種類前後の文字図柄で代用する。

- 7
- BAR
- ベル
- チェリー
- メダル
- マイク
- ロゴ
- 名言

当たり時は最終停止後に対応図柄が揃う。ハズレ時は揃わない。

## あいことば

保存対象:

```js
{
  version: 1,
  coins: 300,
  ownedItems: []
}
```

方針:

- 完全な暗号ではなく、簡易エンコードとして扱う
- JSONを文字コード列に変換する
- ひらがなテーブルで本体コード化する
- 簡易チェック値を末尾に付ける
- 飾りワードをランダム位置に混ぜる

飾りワード:

- まあね
- としやいずむ
- おじさん

復元時:

1. 空白、改行、記号を取り除く
2. ひらがな以外を取り除く
3. 飾りワードを長い順に取り除く
4. 本体コードを読む
5. チェック値を検証する
6. 正しければ保存データを反映する

## 重要な制限

- 現実のお金と結びつけない
- 換金、景品交換、課金を入れない
- ランキングを入れない
- サーバー保存を入れない
- 不正対策は簡易チェックまでにする
